This a simple project to experiment with OpenGl.

OpenGL and Glut required to run.

Visual Studio nuGET can be installed through Package Manager with the command "Install-Package nupengl.core".

Keyboard Commands: 

W - Move up. 
A - Move left.
S - Move down.
D - Move right.
Q - Turn left. 
E - Turn right.
Z - Zoom in.
X - Zoom out.
